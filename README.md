# 🦜 LinguaBird — Cute Multilingual Translator with Voice

LinguaBird is a beautiful AI-powered multilingual translator built with Python and Gradio. It can instantly translate text between 20+ languages and speak the translated text aloud using realistic text-to-speech voices.

Designed with a cute modern UI, animated effects, and real-time audio playback, LinguaBird makes language translation fun, interactive, and beginner-friendly.

## ✨ Features

* 🌍 Translate between 20+ languages
* 🔍 Automatic language detection
* 🔊 Text-to-Speech (TTS) audio playback
* 🇬🇧 British & regional accents support
* 🐢 Optional slow speech mode
* 💅 Beautiful animated pastel UI
* 📋 Copy translated text instantly
* ⚡ Fast real-time translation
* 🦜 Cute mascot-themed interface

## 🛠️ Technologies Used

* Python
* Gradio
* deep-translator
* gTTS (Google Text-to-Speech)
* langdetect
* HTML/CSS animations

## 🚀 Installation

```bash
pip install gradio deep-translator gTTS langdetect
```

## ▶️ Run the App

```bash
python app.py
```
